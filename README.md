
# Movies

 Get latest, popular, and upcoming movies.
## Technologies

**Language:** Swift  
**UI Framework:** UIKit  
**Persistent Framework:** CoreData  
**Design Pattern:** MVVM  
**Testing Framework:** XCTest

## Features

- Get latest, popular, and upcoming movies
- Get movie details
- Local storage of data
